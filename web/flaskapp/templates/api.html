Rail Nation Database API

Content of the database can be retrieved via API. API requests should be prefixed with api and the API version. Current version is v1, so the root of the API is at /api/v1.

Required language should be specified by the two-letter language code. Currently only UK English language is supported, specified by en in the request URL.

Following resources are available

  • Regions /api/v1/en/regions: List of regions.
  • Cities /api/v1/en/<region>/cities: List of cities in a region. Use internal name of the region as <region>.
